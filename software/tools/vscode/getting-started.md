# Getting Started with vscode

```shell
code .
```

## Shortcuts

- ctrl + shift + b = toggle side bar

## Command Palette

- CTRL + SHIFT + P


- CTRL + P = search for files, keep pressing P to navigate between recent files


```shell
:80 # go to line 80
?   # to see help for other commands
@   # list the definitions and move among them
```
