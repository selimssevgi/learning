# 4 elements of jiu-jitsu

- main objective survival, defense


1. defend
  - big (chock)
  - small (keeping the arm away from neck)
  - deep

2. escape
  - accept
  - create
  - force

3. control
  - closed (using strength)
  - open (using body weight positioning)
  - dynamic

4. submit
  - accept
  - create
  - force
